# 常用斜杠命令 {#sec-slash-commands}

Claude Code 提供了一系列斜杠命令（Slash Commands），可以帮助你更高效地管理会话和工作流。以下是最常用的几个命令。

## /resume：查看与恢复历史会话

```bash
/resume
```

Claude Code 会列出你最近的对话历史，你可以选择恢复之前的某个会话，继续之前的工作。

这在以下场景特别有用：

- 昨天进行到一半的数据处理任务，今天想继续
- 之前的工作结果需要微调或扩展

## /btw：运行时旁注对话

```bash
/btw 你觉得刚才生成的代码还可以怎么优化？
```

`/btw` 允许你在 Claude Code 运行过程中插入对话，而**不打断当前进程**。你的问题和回答完成后会自动消失，不会污染主上下文。

这在以下场景特别有用：

- AI 正在执行一个耗时较长的任务，你想顺便问它一个问题
- 不确定 AI 正在做什么，想快速了解一下进度

## /branch：创建分支

```bash
/branch
```

在当前工作状态下创建一个 Git 分支。这对于在不确定方案是否可行的情况下进行实验很有用——如果方案失败，可以轻松回退。

## /rewind：回退

```bash
/rewind
```

将对话状态回退到之前的某个节点。如果你发现 AI 走了弯路，可以用这个命令回到之前的状态重新开始。

## /export：导出会话记录

```bash
/export
```

将当前会话的完整记录导出为文件。这对于以下场景很有用：

- 保存工作日志
- 与同事分享你和 AI 的交互过程
- 将 AI 的输出结果归档

## /insights：使用习惯分析

```bash
/insights
```

生成一份 HTML 报告，分析你过去一个月使用 Claude Code 的习惯，包括：

- 你最常用的操作类型
- 哪些任务耗时最长
- 工作效率的变化趋势
- 改进建议

这个命令可以帮助你反思和优化自己的 AI 协作方式。

## 命令速查表

| 命令 | 功能 | 使用场景 |
|------|------|----------|
| `/resume` | 恢复历史会话 | 继续之前未完成的任务 |
| `/btw` | 运行时旁注对话 | 不打断进程的快速提问 |
| `/branch` | 创建 Git 分支 | 实验性操作前创建安全网 |
| `/rewind` | 回退对话状态 | AI 走了弯路时回退重来 |
| `/export` | 导出会话记录 | 保存工作日志或分享 |
| `/insights` | 使用习惯分析 | 了解和优化自己的使用方式 |
